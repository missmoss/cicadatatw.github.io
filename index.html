About Project Afternoon

Project Afternoon/Project Night 是美國常見的專業/技術社群活動，歡迎對主題有興趣、任何程度與背景的參加者，現場設有各種主題小組讓參與者選擇，亦有初學者小組提供簡單的協助，以及空桌可自由發起主題，我們認為資料新聞的跨領域特質，非常適合這種開放交流、直接組隊一起打魔王的活動方式，在實戰中互相學習成長。

四桌同時並行

R 初學者桌

程式初心者的瑞士刀

R 目前資料分析主流工具之一，免費、開源、資源多，對沒有技術背景的初學者來說比較友善。抓取網路資料、整理、分析到視覺化都難不倒 R，目前臺灣常看到的資料新聞都是使用 R 分析與製圖。我們的 R 講師將帶初學者們使用台灣 R 社群創辦人 Wush Wu 製作的 R 語言翻轉教室來簡單認識 R。

Python 資料分析桌

處理大量資料的好幫手

Python 是資料分析人員常用的工具/程式語言之一，語法簡潔、好上手、可讀性高、開發速度快。從抓取網路資料、整理資料、使用統計及機器學習進行分析到資料視覺化、製作網站，都可以使用 Python 完成。在一般使用、資料相同的狀況下，撰寫 Python 程式可能需時較短、處理的時間也較短。

d3.js 資料視覺化桌

使用圖表與互動呈現內容

D3.js 是在網頁上視覺化資料的利器，能製作出精美的互動圖表，廣泛為國際各大媒體及資料分析產業使用。D3.js 的創造者 Mike Bostock 曾任職紐約時報，使用 D3.js 為紐時製作了許多經典的資料新聞作品。
